# 풍림몰 웹사이트 개발 가이드

## 🔄 최근 변경사항 (2024.12.30)

### 1. Supabase 임시 비활성화

현재 Supabase 인증 없이도 개발 서버가 실행되도록 수정되었습니다.
클라이언트로부터 Supabase 계정 정보를 받으면 쉽게 복원할 수 있습니다.

#### 수정된 파일들:
- `app/root.tsx` - 환경 변수 검증 주석 처리
- `app/core/layouts/navigation.layout.tsx` - Mock 데이터로 대체
- `app/core/lib/supa-client.server.ts` - Placeholder 값 사용
- `app/core/lib/supa-admin-client.server.ts` - Placeholder 값 사용

#### Supabase 복원 방법:
1. `.env` 파일에 실제 Supabase 환경 변수 설정
2. 위 파일들에서 `TODO` 주석을 찾아 원래 코드로 복원
3. 주석 처리된 부분 해제

### 2. Resend (이메일 서비스) 임시 비활성화

Resend는 해외 서비스이므로 국내 프로젝트에 적합하지 않아 임시로 비활성화했습니다.
Mock 클라이언트로 대체되어 실제 이메일이 발송되지 않고 콘솔에 로그만 출력됩니다.

#### 수정된 파일:
- `app/core/lib/resend-client.server.ts` - Mock 클라이언트로 대체

#### 국내 이메일 서비스로 교체 방법:
**자세한 가이드**: `EMAIL_SERVICE_GUIDE.md` 파일 참고

추천 서비스:
- **알리고** (SMS, 알림톡 통합) ⭐ 추천
- 네이버 클라우드 SENS
- 카카오 알림톡
- 토스페이먼츠

영향받는 기능:
- 문의하기 폼 (`app/features/contact/screens/contact-us.tsx`)
- 환영 메시지 (`app/features/cron/api/mailer.tsx`)

## 📦 더미 데이터

모든 페이지에 실제 콘텐츠 대신 더미 데이터가 채워져 있습니다.

### 풍부한 콘텐츠가 포함된 페이지:
- ✅ **인사말**: 대표 인사말, 시그니처, 히어로 섹션
- ✅ **비전과 철학**: 4가지 핵심 가치, 경영 철학 상세 설명
- ✅ **연혁**: 타임라인 형식의 회사 역사 (1990-2024)
- ✅ **제품**: 이미지 포함 제품 카드 (Unsplash 이미지)
- ✅ **뉴스**: 6개 뉴스 아이템, 카테고리별 구분
- ✅ **FAQ**: 4개 카테고리, 아코디언 UI
- ✅ **채용**: 6개 포지션, 복지 혜택, 지원 방법 안내

### 이미지 출처:
- Unsplash API를 사용한 무료 고품질 이미지
- 실제 이미지로 교체 가능

## 🚀 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

서버가 실행되면 `http://localhost:5173` (또는 표시된 포트)에서 확인 가능합니다.

## 🛠️ 기술 스택

- **Framework**: React Router v7
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: Radix UI
- **i18n**: react-i18next
